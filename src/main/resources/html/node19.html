We can use Lamp to emulate this process. For instance, if we wanted the product

we would use Lamp to create a matrix, putting in all the rows of into the first row. The other rows would be filled with repeated columns of each column of as in our examples. Then we would use column operations to accomplish the remaining calculations. Column operations are the same as row operations except that, of course, they act on columns. To get Lamp to use column operations, use the mouse to depress the arrowed row button in the Operation Editor. The button will change to an arrowed column button. Use Operation (2) and Operation (3) as appropriate. Note that Operation (2) does not let you multiply by a zero scalar. You must accomplish this kind of multiplication by selecting all entries of a column which has a zero in the first row entry and setting them to zero. You can do this very easily by sending the zero to the Scalar Editor with the right mouse button, selecting the entries with the left mouse button and then clicking the Enter button in the Scalar Editor.

Lamp is limited to creating matrices so it can only be used for multiplying small matrices. However, if you were stuck on a desert island without enough bamboo and coconuts to build a computer and if you had a burning need to multiply two matrices, you should now be able to do so.
