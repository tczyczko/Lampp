A linear equation in three variables can be drawn as a plane in three-dimensional space. We add the -coordinate to the and coordinates. This is easy to visualize. A box has width ( -axis), height ( -axis) and depth ( -axis). A two-dimensional representation of these three coordinates is given in Figure 1.12. A solution of a system of linear equations in three variables can be thought of geometrically as the point or points where all three planes meet. If they don't have points in common, we don't have a solution. A system of equations that has no solutions is called inconsistent. If it has one or more solutions then it is called consistent.

Geometry fails us when we deal with more than three variables. In fact it is not uncommon for people to deal with thousands of linear equations in thousands of variables. Linear algebra is used to discover if solutions to these systems exist, if a solution which exists is unique and how to find all the solutions.
