Row Operations and Lamp

When a matrix is created using Lamp, an Operation Editor is opened in a window. It is used to execute one of the operations on the rows selected with the right and left mouse buttons.

In the matrix display, the matrix is indexed by the row of numbers in arrow-shaped boxes above and to the left of the entries. By using the left mouse button to click on one of the left arrow indices, that row will become blue to indicate that it has been selected. The corresponding number of that row will be transferred to arrow boxes in the Operation Editor. By clicking the right mouse button on a row arrow, that row will become gray and the associated row arrow in the Operation Editor will be given that row's index. The radio buttons to the left of the operations are used to select which operation to perform when the Execute button is clicked.

The scalar entries in the Operation Editor can be modified in the same way as the entries in the displayed matrix. Left click on one of these entries and a frame will appear around it. Then any value shown in the Scalar Editor can be sent to it by clicking on the Enter button.

There are other ways to select entries for input from the Scalar Editor. Try clicking on the menu bar in the Scalar Editor to see a list.

You can undo the last row or column operation using the Undo button immediately after executing the operation. It is not possible to step back through your row or column operations.

In the menu item for MATRIX in the main display, there are options for STORE, RECALL and SWAP. When STORE is chosen, the values of the displayed matrix are saved and can be returned using the RECALL option. The SWAP option will swap the stored value with the displayed value.



Exercises 2.2

  1. Use Lamp to follow the example of Gauss-Jordan elimination given in this chapter.
  2. Use Lamp to find the solution of the example using less row operations.
  3. The original equations in the example had only integer coefficients and integer constant terms. The solution values were also integers. Can you find a way to solve the system that doesn't involve any pure fractions (rationals that are not integers)?
  4. Use Lamp to graph each step of the example. Then graph each step of the quicker method you found. What can you say about the difference between them?
