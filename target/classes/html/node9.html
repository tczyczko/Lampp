Using Lamp to Graph Linear Equations

We have seen that we can represent a linear equation with a matrix of one row and three columns. We can graph a linear equation if it is composed of real or rational scalars. We shall do so now.

First access the CD that comes with this book using a favorite browser. Assuming all goes well (or that some time was spent perusing the troubleshooting appendix), the HTML version of this text will appear on the computer screen. Go to this page. Click on the Lamp animation immediately below this line.

Figure 1.8: Matrix menu of Lamp
You will see two windows appear. The largest one is where matrices are displayed (see Figure 1.8). The smaller one is called the Operation Editor and we will not be using it at this time.

On the menu bar at the top, move the mouse cursor over the word MATRIX. Click on the word. A menu selection will appear. Choose NEW.

Figure 1.9: Creating a new matrix
Another window will appear asking you to create a matrix (see Figure 1.9). Click on the number to the right of Rows: and choose a size of . Click on the number to the right of Columns: and choose a size of . Then click the Create button. The window will disappear.
Figure 1.10: The Scalar Editor
A new window called the Scalar Editor will appear (see Figure 1.10). This is used to enter scalar values into the matrix that was created. The default field is the rational numbers.
Figure 1.11: Selecting a scalar for input
A zero, a one and a zero will have appeared in the matrix display area. Put the mouse cursor over one of them and click the left mouse button. A frame will appear around that scalar. This means that it will accept input from the Scalar Editor (see Figure 1.11).

Move the cursor to the Scalar Editor and click the left mouse button on the numerator of the rational number that is displayed there. Use the keyboard to edit the number. You may use the Delete key or the Back Space key to remove the zero that is there. You may also use the mouse cursor to select a region of the number to edit. Numbers from the keyboard are entered into this region. When you have finished putting in a number, click the mouse cursor on the Enter button in the Scalar Editor and the number will be sent to the selected matrix entry to be displayed. Multiple matrix entries may be selected for input. If the mouse cursor is over a matrix entry, that scalar can be sent to the Scalar Editor by pressing the right mouse button.

When you have entered values in the matrix, put the mouse cursor over the word GRAPH on the menu and press the left mouse button. A graph of the linear equation represented by the matrix will be displayed. You may have to adjust the scale of the axes using the zoom buttons at the middle bottom of the graph.

You should now feel free to experiment with graphing different linear equations. You should also check out the and buttons in the scalar editor to see what they do. When you have finished viewing the graph, simply click on the CLOSE button at the bottom right corner of the graph display window. WARNING: Moving to another page in the HTML document will permanently erase all of the entries stored in Lamp.
